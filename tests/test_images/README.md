# Test Images for Slip Verification Tests

This directory should contain test images for integration testing.

## Required Test Images:

1. **real_slip_scb.jpg** - Genuine SCB bank slip
2. **real_slip_kbank.jpg** - Genuine KBank slip  
3. **fake_slip.jpg** - Fake/edited slip for scam detection

## How to Add Test Images:

1. Copy real bank slip images here (anonymize sensitive data)
2. Create a fake slip by editing a real one
3. Ensure images are < 5MB each

## Note:

Test images are gitignored for privacy. Add them manually when running tests.
